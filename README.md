### COMMIT convention

- [FEAT] => new feature
- [DOC] => documentation like README
- [FIX] => bugs correction
- [TEST] => testing
- [REFACTOR] => code cleaning or refactoring
- [STYLE] => only for styling
- [BUILD] => new dependancy like npm package
